Projects

Integrate GitHub Monorepo with AWS CodePipeline

This project showcases how to leverage AWS CodePipeline, a continuous delivery service, to automate the release processes of a software project hosted on a GitHub monorepo. The pipeline is customized to trigger service-specific builds and deployments by reading and interpreting GitHub event payloads using AWS Lambda, ensuring a streamlined and efficient CI/CD workflow.

Migrate On-Premises Workloads with AWS Application Migration Service

In this project, I utilized AWS Application Migration Service (AWS MGN) to facilitate the seamless migration of on-premises workloads to the cloud. The project highlights the process of lifting and shifting physical, virtual, or cloud servers to AWS without encountering compatibility issues or significant performance disruptions, thereby reducing the cost and complexity of cloud migration.
